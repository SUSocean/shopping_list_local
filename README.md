To run this app make sure you have the folowing: Docker, Java 17+ (JDK)

1. Run Docker
2. In console navigate to project folder
3. To create local database run ``` docker compose up ```
4. To start the project run
   - win ``` mvnw.cmd spring-boot:run ```
   - mac/linux ``` ./mvnw spring-boot:run ```
5. navigate to ``` http://localhost:8080 ```
